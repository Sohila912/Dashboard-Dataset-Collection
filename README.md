## 🔧 Dataset Preparation Guidelines

### Common Data Transformation Patterns

#### 1. **Date Parsing & Time Hierarchies**
```
Raw Field          → Processed Fields
Date String        → Date, Year, Month, Quarter, Week, Day of Week
DateTime String    → Time, Hour, Period (Morning/Afternoon/Evening/Night)
```

**Example**: From "2022-03-15 14:30:00" to Year=2022, Month=3, Quarter=Q1, DayOfWeek=Tuesday

#### 2. **Categorical Binning**
```
Continuous Field   → Binned Category
Age (numeric)      → Age Groups (18-25, 26-35, 36-50, etc.)
Income (numeric)   → Salary Slabs (0-5k, 5k-10k, etc.)
Distance (numeric) → Distance Ranges (0-5km, 5-10km, etc.)
```

#### 3. **String Parsing & Splitting**
```
Multi-value Field                    → Individual Records
"Python; Java; JavaScript"           → Row 1: Python, Row 2: Java, Row 3: JavaScript
"Netflix, Amazon Prime, Hulu"        → Split and separate for genre/platform analysis
"John Doe; Jane Smith"               → Individual entity records
```

#### 4. **Missing Value Handling**
- **Clinical Data**: Impute with domain-specific medians or conditional replacement
- **Categorical Data**: Retain "Unknown" or "Not Specified" categories
- **Time Series**: Forward fill or interpolation for continuous trends
- **Survey Data**: Explicit "No Response" category

#### 5. **Aggregation & Summarization**
```
Transaction Level      → Business Entity Level
Order Details          → Order Summaries (Qty, Amount, Date)
Individual Ratings     → Average Rating per Product
Page Views             → Monthly Active Users, Engagement Metrics
```

#### 6. **Calculated Metrics & Derived Fields**
```
Base Metrics                          → Derived Metrics
Profit, Sales                         → Profit Margin % = (Profit / Sales) * 100
Matches, Runs                         → Batting Average = Runs / Matches
Total Paid, Copayment                 → Out-of-Pocket %
```

## 📋 Data Format & Structure

### CSV Datasets
- **Format**: Comma-separated values with headers
- **Encoding**: UTF-8
- **Missing Values**: Represented as empty cells or NaN
- **Examples**: Orders-Sales, HR-Analytics, Diabetes-Prediction

### Excel Datasets (XLSX)
- **Format**: Structured workbooks with multiple sheets
- **Contains**: Raw data, processed tables, lookup tables
- **Examples**: Healthcare-Sample, Airbnb-Data-Analysis

### Excel Binary Format (XLSB)
- **Format**: Microsoft Excel binary format (compressed)
- **Requires**: openpyxl or xlrd Python libraries
- **Examples**: Healthcare-Sample-05.xlsb

### Standard Data Fields (by Domain)

#### **Sales/Finance**
- Order ID, Customer ID, Date, Amount, Quantity, Profit, Category, Region

#### **Healthcare**
- Patient ID, Date of Service, Provider, CPT Code, Amount Paid, Claims, Diagnosis

#### **HR**
- Employee ID, Age, Department, Salary, Job Role, Attrition (Y/N), Tenure

#### **Entertainment/Media**
- Title ID, Title Name, Genre, Director, Release Year, Rating, View Count

#### **Transportation**
- Booking ID, Date, Time, Vehicle Type, Distance, Duration, Status, Rating
